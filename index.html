Four Twelve Project

Four Twelve Project is a local family-friendly restaurant that features Moon Township's own Four Twelve Winery. Our menu consists of locally sourced, farm-fresh ingredients served in a comfortable and casual atmosphere that has been created and built using re-purposed materials. Come grow with us!
